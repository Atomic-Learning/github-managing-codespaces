The page [github.com/codespaces](https://github.com/codespaces) displays a list of your Codespaces, as in the image below.

This view shows the name of each Codespace, which repository it is associated with, what type of machine it is running on, how much data is stored in it, and its current state. By opening the menu of a particular Codespace, you can perform various actions including deleting it, or stopping it if it is active.

Limits to Simultaneous Active Codespaces

GitHub only allows a user to have a limited number of active Codespaces at a time. Trying to open more Codespaces than the limit will result in an error, like the one below.

This error will be displayed on the page github.com/codespaces. This allows you to manage your active Codespaces by stopping or deleting some of them to free up space for new ones.
